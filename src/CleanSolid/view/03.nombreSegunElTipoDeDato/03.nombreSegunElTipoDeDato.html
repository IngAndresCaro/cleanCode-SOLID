Nombres según el dipo de dato

Arreglos - Arrays

Ejemplo

Malo

        const frutas = ['manzana', 'platano', 'fresa'];

Regular

        const listaDeFrutas = ['manzana', 'platano', 'fresa'];

Buena

        const frutas = ['manzana', 'platano', 'fresa'];

Mejor

        const nombresDefrutas = ['manzana', 'platano', 'fresa'];

Booleanos

es(is), tiene(has), puede(can), esto siempre estando en positivo

Ejemplo

Malo

        const abierto = true;
        const escrito = true;
        const fruta = true;
        const activo = false;
        const noValor = true;
        const noVacio = false;

Bueno

        const estaAbierto = true;
        const puedeEscribir = true;
        const tieneFruta = true;
        const estaActive = false;
        const tieneValores = true;
        const estaVacio = false;

Mejor

        const isOpen = true;
        const canWrite = true;
        const hasFruit = true;
        const isActive = false;
        const hasValues = true;
        const isEmpty = false;

Números

min, max, total u of para referirnos a un valor numerico

Ejemplo

Malo

            const fruits = 3;
            const cars = 10;

Mejor

            const maxFruits = 3;
            const minFruits = 3;
            const totalFruits = 3;
            const totalOfcars = 10;

Funciones

deben representar acciones, construidas por el vervo con un sustantivo, explicando lo que hace sin entrar todo lo que hace tecnicamente

Ejemplo

Malo

            createUserIfNotExists();
            updateUserIfNotEmpty();
            sendEmailIfFieldsValid();

Mejor

            createUser();
            updateUser();
            sendEmail();
